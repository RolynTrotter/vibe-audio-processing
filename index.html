How it works

Spectral cross-synthesis chops both files into overlapping Hann-windowed chunks, FFTs each chunk, multiplies the two spectra, and overlap-adds them back. The result tracks B's evolving tone — the classic Max MSP pfft~ sound. The shorter file is looped to cover the longer one.

True convolution convolves the whole signals via one big FFT. If B is a short impulse (a room, a clap), A sounds like it was played in that space.

Either way the whole file is scanned for its peak and turned down so it can never clip. Mono out; mismatched sample rates are matched automatically.
